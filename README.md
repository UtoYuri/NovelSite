# NovelSite
课程设计：在线阅读小说网站的设计与实现

本设计服务器端基于ThinkPHP 3.2.3
PHP 版本， 建议 PHP 5.3 以上环境

### 功能
 - [x] 用户注册/登录/密码找回
 - [x] 书籍检索
 - [x] 排行榜（待添加）
 - [x] 书籍付费（虚拟）阅读
 - [x] 在线阅读
 - [x] 我的书架
 - [x] 留言板（待添加）
 - [x] 站内信（待添加）
 - [x] 后台管理（待添加）
